# MCSR 全类型种子自动输入工具
专为MCSR Ranked速通练习设计，支持多类型开局种子自动获取与一键输入，替代手动复制粘贴，大幅提升速通练习效率。

## 功能特性
1. 自动调用API获取6种主流开局类型种子（宝藏/废门/沙漠神殿/村庄/沉船/随机）
2. 一键触发游戏内种子输入全流程，一键安全退出程序
3. 纯键盘逐字符模拟输入，避免剪贴板粘贴的输入错误
4. 后台预加载下一批种子，保证操作连贯性
5. 严格使用MCSR Ranked排位赛sub12种子库

## 环境准备
### 1. 基础要求
已安装Python 3.8及以上版本（推荐3.10+），确保网络可访问MCSR Ranked API地址。

### 2. 安装依赖库
打开命令行执行以下命令，安装工具所需Python库：
```
pip install requests pynput pyinstaller
```

#### 依赖库说明
|库名|核心作用|
|---|---|
|requests|发送网络请求，获取API种子数据|
|pynput|监听键盘按键、模拟键盘输入|
|pyinstaller|将Python脚本打包为可执行文件|

## 使用方法
### 方式一：使用打包后的可执行文件（推荐）
在release中找到exe文件，或将py脚本打包为独立exe文件：
将RANKED.png图标文件与rankbt.py放在同一目录（无图标可省略-i RANKED.png参数）
命令行执行打包命令：
```
pyinstaller -F -w -i RANKED.png rankbt.py
```
打包完成后，在当前目录的dist文件夹中找到rankbt.exe

1. 双击rankbt.exe
2. 在GUI勾选开局类型
3. 设置启动和退出热键
4. 切换到Minecraft游戏界面，按对应按键操作：
5. 按开始热键练习

**开始热键**：自动完成种子输入全流程，同时预加载下一批种子
**退出热键**：立即终止程序，安全退出

设置保存在文件同一目录下的config.json

### 方式二：直接运行Python脚本
将工具代码保存为rankbt.py文件
双击文件，或命令行进入文件所在目录，执行运行命令：
```
python rankbt.py
```

1. 在GUI勾选开局类型
2. 设置启动和退出热键
3. 按开始热键练习
4. 切换到Minecraft游戏界面，按对应按键操作：

**开始热键**：自动完成种子输入全流程，同时预加载下一批种子
**退出热键**：立即终止程序，安全退出

设置保存在文件同一目录下的config.json

## API 相关说明
核心调用API：工具调用MCSR Ranked v2版API，根据选择的开局类型获取对应随机种子：
```
http://43.143.231.104:8001/api/v2/seed?overworld={类型标识}&completion=720000
```

### API 参数说明
|参数|取值|说明|
|---|---|---|
|overworld|对应开局类型标识|指定主世界种子类型，为MCSR Ranked支持的合法类型|
|completion|720000|限定为排位模式种子，且原比赛完成时间≤720000毫秒（12分钟）|

### 完整API文档
MCSR Ranked API v2 官方文档：

## 注意事项
1. 运行程序前，需将Minecraft窗口置于可操作状态，确保为当前活动窗口
2. 种子输入过程中有微小字符延迟（0.01秒/字符），为保证输入准确，请勿在执行过程中手动操作键盘/鼠标
3. 若程序无响应，优先按退出热键
4. 打包可执行文件时，确保文件名无中文/特殊字符，建议以管理员身份运行命令行
5. 工具仅可用于MCSR Ranked速通练习，请勿用于违规游戏场景

## 常见问题排查
### Q1：按下启动无反应？
**A1**：
① 确认Minecraft窗口为当前活动窗口；
② 检查程序是否正常运行（日志无报错）；
③ 关闭其他占用键盘的软件（如输入法、按键精灵等）。

### Q2：种子输入出现缺字/错字？
**A2**：
① 确保游戏内输入框为英文输入法状态；
② 不要在程序执行时手动操作键盘，等待输入流程完成。

### Q3：打包exe时提示报错？
**A3**：
① 升级pyinstaller到最新版本：`pip install --upgrade pyinstaller`；
② 确保文件路径无中文/特殊字符；
③ 以管理员身份运行命令行重新打包。

### Q4：API获取种子失败？
**A4**：
① 检查网络连接，能否正常访问API地址
② 检查API地址是否正确
